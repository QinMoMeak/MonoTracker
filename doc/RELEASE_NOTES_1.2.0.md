# MonoTracker 1.2.0

## 新增
- AI 服务商/模型选择（下拉不可编辑）与“禁用”选项，支持主流厂商统一管理逻辑
- 每个服务商 + 模型独立的 API Key/BaseURL 记忆，不因切换模型而丢失
- Z.ai（原智谱）支持 glm-4.6v-flash，默认 BaseURL：https://open.bigmodel.cn/api/paas/v4/chat/completions
- AI 多图识别：同一商品可选多张图片联合分析
- 心愿单历史价格记录：按日期新增/删除价格，统计展示最新/最低/最高
- 数据管理入口：分类/状态/来源集中在“我的-数据管理”，支持增删改并同步更新已有物品

## 优化
- 添加/编辑物品界面自适应与滚动，键盘遮挡自动居中
- AI 设置与数据管理改为二级弹窗流程
- AI 提示词优化，结果更稳定匹配字段
- 统计面板新增月度支出趋势、渠道分布、分类 Top N
- 搜索与高级筛选（关键词、日期区间、价格区间）
- 统一货币符号与渠道/分类/状态本地化显示
- 导入/导出兼容价格历史字段

## 修复
- 安卓端导出“保存为文件”与“分享”不可用的问题
- 手势返回直接回桌面的问题（添加/编辑/AI 设置/数据管理等）
- 渠道选择重复、货币符号异常显示
- 多语言文本“???”乱码显示
- 分享成功后仍提示不支持的异常提示
