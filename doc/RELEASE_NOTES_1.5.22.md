# Tracker 1.5.22

## 1.5.21 之后的修改汇总

### 国际化与文案修复
- 修复语言切换按钮显示乱码的问题，简体中文、繁体中文、日文、英文统一改为从 `TEXTS` 文案源读取
- 修复部分中文、日文文案损坏问题，补齐并稳定化语言名称相关文案
- 修复 WebDAV 提示文案乱码，避免设置页与弹窗中出现问号或异常字符

### 金额显示修复
- 新增统一金额格式化函数，所有核心金额展示改为通过 `Intl.NumberFormat` 输出
- 修复价格、均价、原价、统计金额、历史价格等位置出现 `\u00a5`、异常前缀或货币符号乱码的问题
- 统一默认 CNY 金额显示行为，减少不同页面之间的格式不一致

### 代码收口与稳定性
- 移除统计页、时间线等位置的硬编码货币符号拼接，改为统一 formatter 管理
- 统计页不再通过 `currencySymbol` 透传金额前缀，减少重复实现与后续维护成本
- 保留输入框中的轻量货币前缀显示，但展示型金额统一走格式化逻辑

### 构建与验证
- Android 版本更新为 1.5.22（versionCode 41）
- 已完成前端生产构建与 Debug APK 构建验证

## 说明
- 本次版本以“乱码修复 + 金额显示统一”为主，属于小范围可直接合并的修复版本
