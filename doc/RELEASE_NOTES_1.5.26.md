# Tracker 1.5.26

## 1.5.25 之后的修改汇总

### Material You 动态配色接入
- Android 12 及以上设备新增 Material You 动态配色支持，可读取系统动态色并同步到应用主界面
- 应用启动时会优先尝试获取系统动态色；若设备或平台不支持，则自动回退到静态 Material 3 配色方案
- 保留原有浅色 / 深色 / 跟随系统逻辑，不改变现有外观模式的使用习惯

### 主题系统重构
- 主题渲染从固定的蓝色、绿色、紫色等硬编码颜色，调整为基于语义 token 的主题系统
- 新增 Material 3 风格主题变量，包括 primary、surface、surfaceVariant、outline 等核心颜色角色
- 前端主渲染层开始通过 CSS variables 消费颜色，为后续继续迁移剩余界面打下基础

### 高使用频率界面迁移
- 主应用外壳、底部导航、主要弹窗、时间线卡片、统计卡片、筛选区和主要按钮已切换到新的主题变量体系
- 主按钮、激活态 chip、导航激活态与主要强调色现在会跟随动态主题变化
- 卡片与表单输入区的表面色和边框色已统一到新的 Material 3 语义层级，整体观感更一致

### Android 原生桥接
- 新增 Android 动态主题插件，用于将系统动态色桥接给 React 前端
- React 前端新增 theme service，负责动态色加载、静态回退和 CSS 变量写入
- 在 Android 上完成动态主题插件编译验证，确保与当前 Capacitor 架构兼容

### 构建与验证
- Android 版本更新为 1.5.26（versionCode 45）
- 已完成前端生产构建与 Debug APK 构建验证

## 说明
- 本次版本属于“主题系统升级的第一阶段”，重点是先完成动态色桥接与高频界面迁移；少量低频界面和零散硬编码颜色后续可继续收口
