# Tracker 1.5.32

## 本次更新

### 国际化文案编码稳定化
- 将 `constants.ts` 中的中文、繁体中文和日文文案统一转换为 `\uXXXX` 形式保存，避免终端、编辑器或构建链路再次错误转码。
- 保持所有语言文案含义不变，但把源码层从“依赖真实 CJK 字符显示”改为“依赖稳定 Unicode 转义”。
- 语言切换、快速开始、AI 设置、WebDAV、筛选面板、统计页等关键区域的文案现在具备更强的跨环境稳定性。

### 文案层风险继续压低
- `constants.ts` 当前非 ASCII 字符计数已降为 0，后续即使在编码环境不稳定的机器上编辑，也更不容易再次污染中日文本。
- 保留此前已经修复的 `lastUsedDate`、`daysUsed`、语言名称和 WebDAV 提示等关键字段。
