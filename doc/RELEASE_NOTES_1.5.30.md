# Tracker 1.5.30

## 本次更新

### 分类业务色与 Material You 兼容
- 将分类颜色从固定 Tailwind 颜色类改为配置里的业务 accent 色值，并在运行时生成图标、徽标和进度条样式。
- 保留数码、服饰、家居、美妆、图书、运动、健康等分类的识别色，但不再让这些固定色直接参与整体主题体系。
- 分类色现在只作为点缀层使用，主界面背景、表面层、按钮、文本仍由 Material You / Material 3 token 统一控制。

### 统计页与时间线的分类展示收口
- 时间线图片角标的分类徽标改为运行时业务色样式，减少与动态主题的冲突。
- 统计页中的分类图标底板和分类占比进度条改为业务 accent 色渲染，不再依赖硬编码的 Tailwind 调色板类名。
- 分类选择区域继续保留原有识别度，但在非选中状态下更贴近当前主题表面层。

### 列表筛选面板主题补齐
- Owned 与 Wishlist 页的搜索图标、高级筛选标签统一切到 Material 3 语义文本色。
- 高级筛选面板的日期、价格输入区域与主题 token 进一步统一，减少旧灰阶残留。
