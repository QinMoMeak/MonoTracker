# Tracker 1.5.27

## 1.5.26 之后的修改汇总

### Material You 第二阶段迁移
- 继续将界面从旧的灰阶/固定色类迁移到 Material 3 语义 token，进一步减少页面中对 `gray/slate` 的直接依赖
- 统一补充表面层、弱强调层、进度条轨道、时间线分割线等语义样式，界面层级更稳定
- 保持 Android 12+ 动态色优先、其他平台静态 Material 3 回退的整体机制不变

### 设置页与弹窗统一
- 设置页中的语言切换、外观模式、导出区、AI 设置、数据管理、WebDAV 面板等区域继续切换到语义化主题类
- 导出弹窗中的按钮、文本容器、说明信息与操作入口进一步统一到新的主题体系
- 错误边界、表单输入、管理项标签等界面元素也改为使用统一的主题表面和 chip 样式

### 列表与统计页细化收口
- 时间线中的数量标签、均价标签、渠道标签、日期标签和状态标签进一步切换到主题 chip 样式
- 统计页中的卡片、TOP 筛选按钮、时间线状态标签、各类进度条底色与辅助线改为语义 token 驱动
- Owned/Wishlist 页的筛选面板和筛选输入框继续去除旧固定背景色，提升动态色下的一致性

### 构建与验证
- Android 版本更新为 1.5.27（versionCode 46）
- 已完成前端生产构建与 Debug APK 构建验证

## 说明
- 本次版本属于 Material You 主题迁移的第二阶段，重点是继续收口高频界面中的旧样式引用；仍有少量低频文本色和分类色块保留原实现，后续可继续渐进迁移
